mikeleung.ca
============

This is my personal website running on Gatsby, React and Netlify.

[Demo](https://www.mikeleung.ca)


Installation
============

To run locally

```

> npm install -g gatsby-cli

> git clone https://github.com/mkleung/mkleung.github.io.git

> npm install

> gatsby develop

> Go to http://localhost:8000

```
